Dynamic exposure

Style based outfits

Outfits can be generated by style and purpose. Your users can select what they plan to use the outfit for and what purpose (ie. special occasion, casual, office etc). Our algorithm sorts out a customized outfit based on the end-users needs.

Intelligent coloring

Trends and personal preferrence are important factors when people buy clothes online. Studies has shown that by exposing garments with season colors sell X % more. By using outfitted your assortment is exposed using the current trends dominant colors and matched using our proprietary outfit-colorizing algorithm.

Generated and procedural outfits

Outfits can be generated from certain garments or garment types. By letting the user affect the input parameters of a generated outfit, it is possible to provide a flow where the user sequentially narrows down to a completely custom outfit.
