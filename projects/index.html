Projects

A selection of open source projects I’ve built. All written in Go unless noted otherwise.

Domainbadger

CLI tool for brute force domain name lookups. Feed it a wordlist and a set of TLDs, and it checks availability in bulk. Originally developed for entirely nefarious purposes.

Börsdata MCP

An MCP server for querying the Börsdata API — Swedish stock market data accessible directly from AI tools.

Yahoo Finance MCP

MCP server that exposes Yahoo Finance data. Same idea as Börsdata MCP, but for the global market.

Zwiki

A Zettelkasten-style wiki built with JavaScript. A lightweight tool for connected note-taking.
