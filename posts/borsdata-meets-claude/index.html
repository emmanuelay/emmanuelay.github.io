Börsdata meets Claude

After building yahoo-finance-mcp last month, I wanted to do the same thing for Nordic data. I’ve been using Börsdata for a while to keep tabs on Swedish stocks, so it was the obvious next target.

The result is borsdata-mcp, an MCP server written in Go that gives Claude direct access to the Börsdata API. Stock prices, financial reports, KPIs, insider transactions, dividend calendars — all queryable through conversation.

Setting it up is straightforward. Download the .mcpb file from the latest release and double-click it, or install via Homebrew:

brew tap emmanuelay/tap
brew install borsdata-mcp

You’ll need a Börsdata API key — some features require Pro+ access.

What makes this interesting is how natural it becomes to explore market data. Instead of clicking through dashboards, you just ask. “Show me insider buys for the last month.” “Compare margins for these three companies.” Claude handles the API calls behind the scenes.

The project is open source and available on GitHub.
